# CyberHome FAQ & Product Knowledge API

📚 一站式 FAQ 和产品知识库 API，支持中英文搜索，可集成到聊天机器人、Shopify 等平台。

## ✨ 功能特性

- ✅ FAQ 智能搜索（31+ 条常见问题）
- ✅ 产品信息搜索（169+ 个产品）
- ✅ 产品描述检索（1000+ 条描述片段）
- ✅ 统一搜索（同时返回 FAQ 和产品）
- ✅ 产品详情查询
- ✅ MCP 协议支持（可接入 LobeChat）
- ✅ 中英文混合搜索
- ✅ 智能关键词扩展（同义词匹配）
